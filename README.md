### Stock market prediction
##### Implementation for several financial instruments - NasdaqGS Price (NFLX, FB, AMZN, TSLA, GOOG, AAPL)

Technologies:
- Python 3;
- TensorFlow 2, Keras;
- pandas;
- pandas_datareader;
- numpy;
- scikit-learn.

Data source:
finance.yahoo.com
